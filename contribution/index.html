In order to submit your changes you will need to create a Pull Request in the GitHub repository. This is not hard at all!

Site structure

  • assets various assets used on the website (icons, site components, etc.)
  • docs all pages related to the Documentation tab
  • contribution all pages related to the Contribution tab
  • about_authors all pages related to the About authors tab
  • scripts
  • styles the site uses consistent theme throughout all pages
  • index.html the website root (currently redirects to docs/index.html)
  • deploy.py script that must be executed using Python 3.9+. The script includes specified html files from assets/site_components into other html files, so developers don't have to modify common parts of the site in all files. As well as temporarily purging them to make files look cleaner.

Editing existing pages

If you found a typo, error or a better solution you can edit a page, by opening the file by the same path as the link you are on.

For example, if you're at https://andromeda-team.github.io/docs/raw_json_text_format.html#page url, then the file you're looking for is located at /docs/raw_json_text_format.html

Then use Ctrl+F to find what you want to change in file and pull request it!

Adding new articles for documentation

Even though site's HTML is well-supported by our own CSS, making it easy to write, you can submit plain text or markdown in your pull requests, then one of the site maintainers will format & put together everything for you! Though, make sure your article is well-written beforehand
