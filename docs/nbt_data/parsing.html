NBT Data parsing

Unfortunately in minecraft you are unable to parse any sorf of dynamic variable into a command, that's when custom solutions take place. This article is going to describe all efficient algorithms related to this topic.

Getting n-th element

If you need to get an n-th element of an array (Example: data get entity @s Inventory[n] where n is scoreboard players get @s name ), then this algorithm should work for you.

Setting properties (optional) You can set variables to your personal ones, to get the best experience out of explanation. You will also be able to simply copy & paste the code into the coresponding functions!

Function namespace:my/path/parse:

  1. 1. We need to create a temporary data holder, which we can modify without losing any data:
  2. data modify storage minecraft:temp Instance.Parsing set from entity @s Inventory

  4. # 2. We create a counter, that will start with the size of the array and decrease in a loop, as well as a result placeholder:
  5. execute store result score #Counter my_scoreboard run data get entity @s Inventory
  6. data remove storage minecraft:temp Instance.Match

  8. # 3. Then if the array isn't empty we start a loop:
  9. execute if score #Counter my_scoreboard matches 1.. run function namespace:my/path/parse_step

Function namespace:my/path/parse_step:

  1. # Decrement counter
  2. scoreboard players remove #Counter my_scoreboard 1

  4. # 4. We remove the last element of the array
  5. data remove storage minecraft:temp Instance.Parsing[-1]

  7. # Repeat the function or return the field depending on the counter:
  8. execute if score #Counter my_scoreboard matches ..0 run data modify storage minecraft:temp Instance.Match set from storage minecraft:temp Instance.Parsing[-1]
  9. execute if score #Counter my_scoreboard matches 1.. run function namespace:my/path/parse_step

In the end we will have our result by using data get storage minecraft:temp Instance.Match
