/execute is the most powerful command that lets you execute any other command with special parameters, different executers, preconditions and storing the result of the executed command.

The syntax is execute <subcommands> run <command>, where:

  • subcommands is a list of subcommands separated with a space;
  • command is any minecraft command (without any changes to its syntax);
