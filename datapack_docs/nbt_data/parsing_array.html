Parsing an array

Is used when we want to read/modify an element of array (Basically do data get/modify ... Array[n] where n is a variable )

There are a couple of methods, so use tabs above the page to find the one that suits you the best!

Method 1

We are creating a copy of our target, then we get its last element [-1], we can do any checks with this element, then we remove it and go through this copy until it's empty.

File parse.mcfunction
# 1. We need to create a temporary data holder, which we can modify without losing any data: data modify .Parsing set from
# 2. We create a counter, that will start with the size of the array and decrease in a loop, as well as a result placeholder: execute store result score #Counter run data get data remove .Match
# 3. Then if the array isn't empty we start a loop: execute if score #Counter matches 1.. run function parse_step
scoreboard players remove #Counter 1
# 4. We remove the last element of the array data remove .Parsing[-1]
# Repeat the function or return the field depending on the counter: execute if score #Counter matches ..0 run data modify .Match set from .Parsing[-1] execute if score #Counter matches 1.. run function parse_step
Define a path where this algorithm will be executed Define a scoreboard you will be using Define the NBT Path that you will be parsing Define a temporary data holder

Method 2

Is way simpler and way more versatile, it lets you not only read, but even modify the array!

It uses only 1 Array, we get the first element, we can read/modify it, then we move this element to the end of the array and repeat this project until we go through all elements. It is important to create a counter, because otherwise this loop will be infinite.

File parse.mcfunction
execute store result score #Counter run data get execute if score #Counter matches 1.. run function step
scoreboard players remove #Counter 1
# At this line we can do with the element whatever we want, print it, check it, modify and more by using: # data get [0]
data modify append from [0] data remove [0]
execute if score #Counter matches 1.. run function step
