📜 Description

This site gives you access to a large library of solutions and work arounds for advanced data-packers!

Every single page on this site is specialized on the data packs. So don't be shy to explore the categories you might "already know", since you might find something new!

As an example there's a Minecraft Commands category which not only describes the command, but gives it a performance rating, all kinds of outputs (that can be obtained with /execute store command), etc.

🤝 Contribute

Do you think you know a better way to solve a problem? You found a typo? Do you want to suggest a solution to a new problem? You can contribute by creating a pull request

The site's HTML is pretty simple to write, though you don't have to worry about the design too much, since I will make sure your work looks pretty and clean! Read more about contribution on the Contribution page
